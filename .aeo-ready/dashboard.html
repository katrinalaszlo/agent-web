Direct answer formatting
Most pages don't lead with a concise summary (40-60 words). This is what AI systems cite.
+3 pts
1-2 hours
Steps:
  1. On each key page, add a 40-60 word summary as the first paragraph
  2. This paragraph should directly answer the question the page addresses
  3. Don't start with background or context — lead with the answer
  4. AI systems extract this as the citation snippet
Example:
<!-- Before -->
<h1>Our Billing Platform</h1>
<p>Founded in 2020, we set out to solve...</p>

<!-- After -->
<h1>Our Billing Platform</h1>
<p>Acme is a usage-based billing platform that helps SaaS companies meter API calls, automate invoicing, and recognize revenue. Plans start at $0/mo for up to $10K MRR.</p>
